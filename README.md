# <p align="center">🔓 <b><span style="font-size: 24px;">FHC (fckhappcrypt) — Happ Crypt Decryption</span></b></p>

<p align="center">
  <b>FHC</b> is a desktop application for decrypting <b>Happ Crypt</b> subscriptions and managing <b>VLESS/VMess/Trojan</b> configs.<br>
  <i>The project is in the development stage (beta)</i>
</p>

---

## <span style="font-size: 20px;">🚀 <b>Quick Start</b></span>

1. Download or clone the repository  
2. Go to the program folder  
3. Run: <code><b>FHC.exe</b></code>  

> <i>When you first launch, dependencies will be installed automatically.</i>

---

## <span style="font-size: 20px;">🎮 <b>Features</b></span>

- Decrypt <code>happ://crypt...</code> links  
- Download regular HTTP/HTTPS subscriptions  
- Extract <code>vless://</code>, <code>vmess://</code>, <code>trojan://</code> configs  
- View, copy, save to file  
- Subscription history in <code>subscriptions.json</code>  
- Dark theme interface
